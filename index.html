Porquê usar HTML5/CSS3?

Algumas informações sobre as tecnologias

Desempenho

CSS3 é rápido, pois consegue ser renderizado sem processamento de JavaScript e com uso mais eficiente do hardware como placa gráfica.

Compatibilidade

A grande maioria das plataformas para acesso à internet dá suporte a CSS3 e a tendência é cada vez mais se tornar um padrão

Simplicidade

HTML5 e CSS3 são simples de se entender e configurar. Com novas funcionalidades como Flex e Grid, se torna cada vez mais prático.
